TOR Relay (non-exit node)

This is a TOR relay (https://www.torproject.org/) which does NOT act as an exit node (i.e. it relays traffic between nodes in the TOR anonymization network). It should NOT generate traffic to non-TOR machines. No logs that would allow identification/tracing of users are kept.
